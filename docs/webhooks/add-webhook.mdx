---
title: "Add Webhook"
description: "Add a Webhook for an event"
api: "POST /webhooks"
---

Available events:

* `response_summary`

* `collected_info`

* `conversion`

### Parameters

<Param body="event" type="string">
  The event you want to subscribe to
</Param>

<Param body="url" type="string">
  The url to send the event to
</Param>

<RequestExample>
  ```bash Request
  curl -X POST https://api.gotolstoy.com/webhooks \
    --header 'Authorization: Bearer <API_KEY>'
  ```
</RequestExample>
